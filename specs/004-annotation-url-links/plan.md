# Implementation Plan: Clickable URL Links in Annotations

**Branch**: `004-annotation-url-links` | **Date**: 2026-05-15 | **Spec**: [spec.md](spec.md)  
**Input**: Feature specification from `specs/004-annotation-url-links/spec.md`

## Summary

Render `http://`/`https://` URLs in annotation values as clickable links that open in the default browser. Image URLs (`.png`, `.jpg`, `.jpeg`, `.gif`, `.svg`, `.webp`) additionally render an inline image preview. Each annotation row gains display/edit mode: display mode shows linkified content; edit mode (entered by clicking the row) shows the existing editable widget. A new `openExternal` webview message type routes link clicks through `vscode.env.openExternal`. The entity editor's CSP `img-src` is extended to allow `https:` sources for inline image loading.
