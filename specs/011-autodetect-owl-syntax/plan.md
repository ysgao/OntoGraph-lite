## Summary

Extend `detectOwlFormat` in `ParserRegistry.ts` to identify all five OWL serialisation
formats from content, and update the `'owl-xml'` dispatch case to route `.owl` files to
the correct parser. The fix also corrects the Functional Syntax fingerprint from a
file-start `Prefix(` check to a 4 KB scan for `Ontology(`.

## Phase 0: Research

**Status**: Complete. See [research.md](research.md).

Summary of decisions:
- Detection applies only to `'owl-xml'` languageId (`.owl` files)
- Functional Syntax: scan first 4 KB for `Ontology(`
- Manchester Syntax: scan first 2 KB for `Ontology:`
- OWL/XML: `<Ontology` + `owl#` namespace in first 2 KB
- RDF/XML: `<rdf:RDF` in first 2 KB
- Turtle: `@prefix`/`@base`/`PREFIX `/`BASE ` in first 1 KB
- BOM handled by `trimStart()`
- Detection co-located in `ParserRegistry.ts`, not moved to `extension.ts`

## Phase 1: Design & Contracts

**Data model**: See [data-model.md](data-model.md). No persistent state changes. `detectOwlFormat` return type extends to include `'manchester'` and `'turtle'`.

**Contracts**: No public API changes. `ParserRegistry.parseAsync` and `ParserRegistry.parse` signatures are unchanged. `detectOwlFormat` is a private (non-exported) function.

**Agent context**: Updated below.

### detectOwlFormat — revised algorithm

```
Input: raw file text (string)
1. t = text.trimStart()                        // strips BOM + whitespace
2. if t.startsWith('<'):
     head = t.slice(0, 2000)
     if /<Ontology[\s>]/.test(head) AND /owl#/.test(head)  → 'owlxml'
     if /<rdf:RDF[\s>]/.test(head)                         → 'rdfxml'
     return 'unknown'
3. head4k = t.slice(0, 4096)
   if head4k.includes('Ontology(')             → 'functional'
4. head2k = t.slice(0, 2048)
   if head2k.includes('Ontology:')             → 'manchester'
5. head1k = t.slice(0, 1024)
   if /(?:@prefix|@base|PREFIX\s|BASE\s)/.test(head1k)    → 'turtle'
6. return 'unknown'
```

### ParserRegistry.parse — owl-xml case (revised)

```typescript
case 'owl-xml': {
  const fmt = detectOwlFormat(text);
  if (fmt === 'functional')  { model = new FunctionalParser(text, uri).parse();  sourceFormat = 'functional';  break; }
  if (fmt === 'owlxml')      { model = new OwlXmlParser(text, uri).parse();      sourceFormat = 'owl-xml';     break; }
  if (fmt === 'rdfxml')      { model = new RdfXmlParser(text, uri).parse();      sourceFormat = 'rdf-xml';     break; }
  if (fmt === 'manchester')  { model = new ManchesterParser(text, uri).parse();  sourceFormat = 'manchester';  break; }
  if (fmt === 'turtle')      { model = new TurtleParser(text, uri).parse();      sourceFormat = 'turtle';      break; }
  throw new Error(`Could not detect OWL serialisation format for: ${uri}`);
}
```
